# Stack Tecnológico

| Componente | Tecnologia | Versão |
|------------|------------|--------|
| Frontend | Angular | 21 |
| Backend | Node.js + Express | 22 / 5 |
| Linguagem | TypeScript | 5.9 |
| Database | PostgreSQL | - |
| ORM | TypeORM | 0.3 |
| DI | TypeDI | 0.10 |
| Autenticação | JWT | 9.0 |
| Logging | Winston | 3 |

## Backend
- **Node.js 22** com **Express 5**
- **TypeScript 5.9** para type safety
- **TypeORM 0.3** para persistência em PostgreSQL
- **TypeDI 0.10** para inversão de controlo
- **JWT 9.0** para autenticação stateless
- **Winston 3** para logging

## Frontend
- **Angular 21** com foco em UI limpa e rápida
- **Dark Mode** por defeito
- **Signals** para gestão de estado reativo
- **Estrutura de componentes modular**
